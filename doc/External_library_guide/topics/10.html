2.8  FTP file systems

1. The directory containing files of this external library is: installation directory\esProc\extlib\FtpCli. The Raqsoft core jar for this external library is scu-ftp-cli-2.10.jar.

 

2. esProc provides functions ftp_client(), ftp_cd(), ftp_dir(), ftp_put(), ftp_mput(), ftp_get(), ftp_mget() and ftp_close() to access the FTP file system. Look them up in【Help】-【Function reference】to find their uses.
