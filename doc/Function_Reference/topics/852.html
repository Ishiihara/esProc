sf_close()

Description：

Close Salesforce cloud server connected through the way of rest api.

Syntax:

sf_close(fd)

Note:

The external library function (See External Library Guide) closes Salesforce cloud server connected through rest api to release resources.

Parameters:

fd

restclient object

Example:

 

A

 

1

=sf_open()

Connection

2

=sf_close(A1)

Close connection
