hdfs_close()

Description:

Close the HDFS connection.

Syntax:

hdfs_close(hd)

Note:

This external library function (See External Library Guide) closes a connection to the HDFS file system.

Parameter:

hd

An hd connection

Example:

 

A

 

1

=hdfs_open("hdfs://192.168.0.8:9000")

Connect to the HDFS file system

2

=hdfs_close(A1)

Close the HDFS connection
