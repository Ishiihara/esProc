Qdirectory()

Description:

Get a list file names that have the specified prefix from the remote storage.

Syntax:

Qdirectory(path,p,d)

Note:

The function gets a list file names that have the specified prefix from the remote storage. The list should contain file name (Key), number of bytes in the file (Size) and the date when the file is last updated (Date).

When all parameters are absent, return a list of all storage buckets.

When delimiter d is specified for S3 cloud storage type, the function returns all file names before the delimiter appears the next time – that is, return file names without delimiters; default delimiter is ¡°/¡±.

Parameter:

path

Storage bucket name; should not be omitted when both of the other two parameters are absent.

p

Prefix string, which is case-sensitive and can be absent.

d

The delimiter; support different delimiters for different cloud storage services.

Option:

@p

Display the path.

Return value:

List of file names

Example:

Take S3 cloud storage as an example:

 

A

 

1

=Qdirectory("test1","a1/aa1/","")

Get a list of file names that have the prefix ¡°a1/aa1/¡± from storage bucket test1.

2

=Qdirectory()

Return a list of all storage buckets.

3

=Qdirectory@p("un-bucket100")

Get a list of file names that do not contain delimiter ¡°/¡± and are represented by paths from storage bucket un-bucket100.

4

=Qdirectory("test1","a1/aa1/","_")

 

Get a list of file names that have prefix ¡°a1/aa1/¡± and do not contain delimiter ¡°/¡± from storage bucket test1.
