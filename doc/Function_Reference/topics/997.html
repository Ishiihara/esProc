ym_statistics()

Description:

Get a series of information about a specified variable.

Syntax:

ym_statistics(md, field)

Note:

The external library function (See External Library Guide) gets a series of information about a specified variable, including the max & min values, degree of importance, missing rate, and deviation, etc., to facilitate users¡¯ data mingling and statistical analysis process.

Parameter:

md

An MD object

field

Variable name

Return value:

A table sequence
