redis_close()

Description:

  Close a Redis database connection.

Syntax:

  redis_close(con)

Note:

  This external library function (See External Library Guide) disconnects from a Redis database.

Parameter:

con

Database connection string

Return value:

  No return value

Example:

 

A

1

=redis_open()

2

=redis_close(A1)
