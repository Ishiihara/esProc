avro()

Description:

Read and Avro data file.

Syntax:

avro(avrofile,schemafile;Fi...)

Note:

External library function (See External Library Guide).

The function reads an Avro data file. When both the specified field name Fi and the specified file schemafile are present, the former has greater priority than the latter; when both are absent, read all fields in the avrofile file.

Parameter:

avrofile

Avro file object; cannot be omitted.

schemafile

The avsc file object describing data structure.

Fi

Field name.

Option:

@c

Return a cursor.

Return value:

Table sequence/Cursor

Example:

 

A

 

1

=file("person.avro")

Read an Avro file.

2

=file("person.avsc")

Read the avsc file.

3

=avro(A1)

Return all fields of A1’s Avro file.

4

=avro(A1;name,age)

Return the specified fields.

5

=avro(A1,A2)

Read data from A1 according to A2’s data structure.

6

=httpfile("http://localhost:7080/test/person.avro")

Read the Avro file in the HTTP service.

7

=avro(A6)

Return all fields of A6.
