asin()

Description:

Compute arc sine value of the specified data.

Syntax:

asin(number)

Note:

The parameter number is a real number between -1 and 1.

Parameter:

number

The real number for which you want to compute the arc sine value

Return value:

Arc sine value

Example:

asin(-1)

-1.5707963267948966

asin(sin(pi()/2))

1.5707963267948966

asin(sin(0))

0.0

Related functions:

acos()

atan()
