transpose()

Description:

Transpose a matrix.

Syntax:

transpose(A)

Note:

The function transposes matrix A; enter 0 as member values if the matrix has empty members.

Parameter:

A

A matrix

Option:

@n

Enter null as member values if the matrix has empty members

Return value:

Sequence

Example:

 

A

 

1

[[1,1,1],[2,2,2],[3,3,3]]

 

2

=transpose(A1)

Calculate positions of members of matrix A1

3

[[1],[2,2],[3,3,3]]

 

4

=transpose(A3)

Transpose matrix A3; enter 0s to empty members

5

=transpose@n(A3)

Use @n option to enter nulls to empty members
