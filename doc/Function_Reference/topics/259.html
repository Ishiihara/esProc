dynadb._query()

Description:

Query data stored in DynamoDB.

Syntax:

dynadb._query(sql)

dynadb.query(A,sql)

Note:

The external library function (See External Library Guide) queries data in DynamoDB using a SQL statement.

Parameter:

A

A sequence on which the sql statement is executed circularly while calculating different parameter values

to pass to sql for execution

sql

A SQL statement

Return value:

Table sequence

Example:

 

A

1

=dyna_open("http://localhost:8000", "us-east-2")

2

=A1.query("select * from emp")

3

=A1.query("select * from emp where lname in (?,?,?) and gender = ?","Gary","Eddy","Lee","F")

4

=A1.query("select * from emp where gender =?","M")

5

[201,'Joshua',205,'Emma',212,'Ryan']

6

=A1.query(A5, "select * from emp where pid = ? and fname=?")

7

>A1.close()
