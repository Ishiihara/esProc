x+y

Description:

Join two or more strings end-to-end.

Syntax:

x+y

Note:

A string and a numeric value cannot be concatenated.

Parameter:

x

A string constant

y

A string constant

Return value:

A string formed by joining x and y.

Example:

"abc"+"def"

abcdef

"abc"+123

123. A string and a numeric value cannot be concatenated.
