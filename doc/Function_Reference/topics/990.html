ym_performance()

Description:

Get information about the performance of a model.

Syntax:

ym_performance(pd)

Note:

The external library function (See External Library Guide) gets information about the performance of a model, such as Gini index, AUC index and KS index.

Parameter:

pd

A PD model object

Return value:

Table sequence
