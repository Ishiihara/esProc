hbase_close()

Description:

Close a HBase connection.

Syntax:

hbase_close(con)

Note:

This external library function (See External Library Guide) closes HBase connections.

Parameter:

con

Database connection string

Example:

 

A

 

1

=hbase_open("hdfs://192.168.0.8:9000")

 

2

=hbase_close(A1)

Close the HBase connection

Related function:

  hbase_open()
