datetime(datetimeExp)

Description:

Adjust the precision of the datetime expression and return it.

Syntax:  

datetime(datetimeExp)

Note:  

The function adjusts the precision of datetimeExp and returns it. By default, the precision is the day.

Parameter:  

datetimeExp

Datetime value

Option:

@m

Accurate to minute

@s

Accurate to second

@h

Accurate to hour

Return value:  

Datetime value

Example:

 

A

 

1

=datetime(now())

2013-12-09 00:00:00

2

=datetime@m(now())

2013-12-09 16:56:00

3

=datetime@s(now())

2013-12-09 16:56:45

4

=datetime@h(now())

2013-12-09 16:00:00
