cosh()

Description:

Return the hyperbolic cosine.

Syntax:

cosh(number)

Note:

The parameter number is a real number.

Parameter:

number

The real number for which you want to find the hyperbolic cosine

Return value:

The hyperbolic cosine

Example:

cosh(4)

27.308232836016487
